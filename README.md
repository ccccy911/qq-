# 基于springboot的qq邮箱定时发送邮件
每天早上8点，定时发送一条二次元美少女发来的活力满满的文案；
每天晚上12点，定时一条网易云热评😭😭😭
内部调用Spark X1-32K的API，自动生成邮件内容
